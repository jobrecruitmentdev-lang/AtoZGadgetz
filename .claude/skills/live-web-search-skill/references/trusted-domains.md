# Trusted Domain Whitelist for Web Engineering Search
When evaluating web snippets or passing filtering flags to your search commands, prioritize extraction from these platforms:

| Target Layer | Primary Trusted Source |
| :--- | :--- |
| **FastAPI Backend** | `tiangolo.com`, `docs.pydantic.dev` |
| **Next.js Frontend** | `nextjs.org/docs`, `react.dev` |
| **ORM & Database** | `prisma.io/docs`, `postgresql.org/docs` |
| **Community Fixes** | `stackoverflow.com`, `github.com` |
